# VoiceLive Mobile

Professional vocal processor for iPhone with iRig mic support. Real-time audio effects processing using Web Audio API.

## Features

- **Real-time Vocal Effects**: Reverb, Delay, Chorus, Compression, Autotune, and 3-band EQ
- **Artist Presets**: 15 famous vocal performer presets (Adele, Freddie Mercury, Billie Eilish, and more)
- **Pitch Detection**: Real-time pitch detection with musical note display
- **Auto-Tune Engine**: Pitch correction with selectable musical keys
- **Hardware-Inspired UI**: TC Helicon-style metallic interface

## Hardware Requirements

- iPhone with iOS Safari
- Lightning to 3.5mm adapter
- iRig mic (or compatible TRRS microphone)

## Development

```bash
npm install
npm run dev
```

## Deployment

This app is deployed on Vercel and optimized for iPhone Safari.

## Tech Stack

- React + TypeScript
- Web Audio API
- Vite
